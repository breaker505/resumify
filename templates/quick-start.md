# 通用快速模板

把这个模板填好发给 AI，就能得到一份完整的《实习提炼报告》。

建议至少填 80% 的信息，越详细输出越准。

---

## 输入模板

```
# 实习经历基本信息

## 基础信息
- 公司行业：
- 岗位名称：
- 实习时长：
- 是否在职：

## 日常工作（越具体越好）
我每天主要做什么：
[写你的日常工作]

## 接触的系统 / 工具 / 产品
我用过哪些工具、系统、或接触过什么产品：
[写你用过的]

## 解决过的问题 / 异常 / 难点
遇到过什么问题或异常，怎么处理的：
[写你处理过的]

## 团队协作方式
- 团队有多少人：
- 和谁配合最多：
- 沟通方式（例会？线上？）：

## 学习 / 产出 / 成果
这段实习有什么成果或学到的东西：
[写你的成果]

## 参与深度
- [ ] 主导完成
- [ ] 协助推动
- [ ] 配合执行
- [ ] 旁观学习

## 目标岗位信息
你打算投什么岗位 / 行业：
[写目标岗位]

## 最担心的点
这段实习你最担心哪方面不好包装 / 说不清楚：
[写你的担忧]
```

---

## 输出预览

填好提交后，你会得到：

```
1. 核心定位        → 这段实习的本质和价值
2. 简历版          → 2-4 条可以直接用的简历 bullet
3. 面试口述版      → 面试时的回答框架
4. STAR 亮点       → 最值得讲的一件事
5. 核心亮点提炼    → 这段经历真正值钱的地方
6. 岗位适配建议    → 不同岗位怎么改写
7. 诚实警告        → 哪些千万别硬讲
```
