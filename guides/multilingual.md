# 中英双语简历指南

> 实习.skill 的「岗位适配」支持中英文双版本输出。
> 如果你是留学生或投外企，这里告诉你关键差异。

---

## 中文简历 vs 英文简历

### 中文简历

| 特征 | 说明 |
|------|------|
| 字数 | 每条 bullet 15-30 字 |
| 语气 | 偏动作感、直接 |
| 结构 | "做了什么 + 怎么做的 + 结果" |
| 要求展示 | 对这件事的掌握程度 |

示例：
> 参与电商中台服务端接口测试与回归验证，独立完成 6+ 轮全量回归，定位并协助排查 4 个影响核心链路的缺陷

### 英文简历

| 特征 | 说明 |
|------|------|
| 字数 | 每条 bullet 80-120 字符 |
| 语气 | 更强调结果和贡献 |
| 结构 | Action verb + Task + Result |
| 要求展示 | 可量化的影响力 |

示例：
> Participated in server-side API testing and regression for e-commerce mid-platform, independently completing 6+ full regression cycles, identifying and assisting in resolving 4 defects impacting core business chains

## 关键词差异

| 中文常用 | 英文对应 |
|---------|----------|
| 参与 | Participated in / Contributed to |
| 负责 | Responsible for / Led |
| 协助 | Assisted / Supported |
| 独立完成 | Independently completed |
| 优化 | Optimized / Enhanced |
| 提升 | Improved / Increased |
| 推动 | Drove / Facilitated |
| 版本迭代 | Version delivery / Release cycle |

---

## 常用 Action Verb

| 领域 | 推荐动词 |
|------|---------|
| 开发 | Developed, Built, Implemented, Architected |
| 测试 | Validated, Tested, Verified, Debugged |
| 数据 | Analyzed, Processed, Modeled, Cleaned |
| 产品 | Designed, Defined, Shipped, Prioritized |
| 运营 | Managed, Grew, Optimized, Coordinated |
| 设计 | Created, Prototyped, Redesigned, Iterated |
| 销售 | Generated, Closed, Expanded, Negotiated |

---

## 使用方式

在提交实习内容时，**在模板中注明需要中英文双版本**。

实习.skill 会输出两份简历版 + 岗位适配时也会从语言角度给出建议。
