# 简历优化指南

> 实习.skill 的「简历版」输出可以直接用，但简历优化还有很多细节。

---

## 一页原则

大多数校招和实习简历应该**控制在一页 A4 以内**。

实习.skill 每条 bullet 控制在 15-35 字（中文），就是为这个目标服务的。

## Bullet 结构

### 不是写清单，是写贡献

❌ 不好的写法：
> 负责设备测试、异常排查、文档整理

✅ 好的写法（实习.skill 风格）：
> 参与电商中台服务端接口测试与回归验证，独立完成 6+ 轮全量回归，定位并协助排查 4 个影响核心链路的缺陷

### 公式

```
做了什么 + 怎么做的 + 结果/数据
```

## 排序策略

如果简历上有 2-3 段实习，建议：

1. **最相关 + 最有亮点** → 第一条、写最详细
2. **相关但略平淡** → 第二条、2-3 条 bullet
3. **完全不相关** → 可能只写一行或不写

## 和实习.skill 配合

| 步骤 | 说明 |
|------|------|
| Step 1 | 跑实习.skill 获得完整提炼报告 |
| Step 2 | 从「简历版」选取 2-4 条 bullet |
| Step 3 | 放入简历中对应公司的条目下 |
| Step 4 | 根据「岗位适配建议」做关键词微调 |
| Step 5 | 根据「诚实警告」准备面试回答 |
